Create Parent Student Relation

Cria a relacao entre um parente e um estudante. Para que esse método possa ser executado e concluido com sucesso, é necessário que ja exista um parente criado. Essa requisição não cria parentes, estudantes ou graus de parentesco.

  • Curl
  • HTTP
curl -X POST -H "Content-Type: application/json" -d '{
	"student": {
		"id": 1
	},
	"parent": {
		"id": 2
	},
	"kinship": {
		"id": 2
	}
}' "http://54.233.195.52:8080/parent_students"
POST /parent_students HTTP/1.1
Host: 54.233.195.52:8080
Content-Type: application/json

{
	"student": {
		"id": 1
	},
	"parent": {
		"id": 2
	},
	"kinship": {
		"id": 2
	}
}

Create User Parent

  • Curl
  • HTTP

	"name": "Parente de Teste 2",
	"email": "parente2@escolando.com",
	"password": "teste"
}' "http://54.233.195.52:8080/user_parent"

Content-Type: application/json

{
	"name": "Parente de Teste 2",
	"email": "parente2@escolando.com",
	"password": "teste"
}

Create Parent User And Student

Cria um estudante, um usuario e um perfil de parente, no qual representa o responsável do novo estudante criado.

Em outras palavras cria o responsável legal (e seus devidos acessos) e o estudante ao mesmo tempo.

  • Curl
  • HTTP
curl -X POST -H "Content-Type: application/json" -d '{
	"student": {
		"name": "Estudante Filho do Testerson",
		"institution": {
			"id": 1
		}
	},
	"parent": {
		"name": "Testerson",
		"email": "testerson@gmail.com"
	},
	"kinship": {
		"id": 1
	}
}' "http://54.233.195.52:8080/user_parent_and_student"
POST /user_parent_and_student HTTP/1.1
Host: 54.233.195.52:8080
Content-Type: application/json

{
	"student": {
		"name": "Estudante Filho do Testerson",
		"institution": {
			"id": 1
		}
	},
	"parent": {
		"name": "Testerson",
		"email": "testerson@gmail.com"
	},
	"kinship": {
		"id": 1
	}
}
